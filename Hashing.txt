The Division Method

To fit a huge number K into a small array of size M, we use the modulo operator:

h(K)=KmodM
Example: If your table size is 10 and your key is 157, the index is 157mod10=7.

Collisions: "The Roommate Problem"

A collision occurs when two different keys result in the same index.

Example: Both 17mod10 and 27mod10 want to go to Index 7.

How to handle Collisions?

Chaining (Used by C++ STL): Each index in the array doesn't hold a single value; it holds a Linked List. If multiple keys hit Index 7, they just get added to that list.

Why O(n) Worst Case? If your hash function is bad (or someone gives you "collision-prone" data), every item might end up in the same list at Index 7. Searching that list takes O(n).

üî• Interview Tip: When to avoid unordered_map
While it should be your first choice, mention that in competitive programming, some test cases are designed to trigger the O(n) worst case (Hash Collisions). In those rare cases, switching to a regular map or using a custom hash function is the fix.
